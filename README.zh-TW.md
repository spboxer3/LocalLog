# ⏱️ LocalLog - 簡單隱私的網頁紀錄器


這是一個輕量級且注重隱私的 Chrome 擴充功能，幫助您追蹤瀏覽習慣、設定每日時間限制並提升工作效率。所有數據皆僅儲存在您的本機裝置上，絕不外傳。


## ✨ 主要功能


### 📊 數據分析儀表板


- **即時追蹤**：僅在視窗處於焦點且分頁活動時進行計時，確保數據精確。
- **視覺化圖表**：透過互動式儀表板查看日、週、月的活動趨勢。
- **分類統計**：自動分類熱門網站（工作、社群、影音等），亦支援自訂分類。
- **熱門網站**：一目了然地檢視您的時間都花在哪裡。


### 🛡️ 生產力工具


- **每日限制**：為特定網域或整個分類設定每日使用上限（分鐘）。超時將發送通知提醒。
- **專注模式**：開啟「專注模式」後，當您訪問非工作類網站時會收到警告。
- **黑名單系統**：支援通配符（例如 `*.example.com`），可將特定網域或子網域完全排除在追蹤之外。


### 💾 資料管理


- **隱私優先**：100% 使用 Chrome 本地儲存（Local Storage）。無後端伺服器，無數據上傳。
- **匯入與匯出**：
  - 匯出瀏覽紀錄為 **CSV** 格式，方便進行外部分析。
  - **新功能**：支援匯入 CSV 資料，可還原紀錄或合併多裝置的數據。
  - 透過 **JSON** 備份與還原您的設定與規則。
- **資料掌控**：一鍵清除所有瀏覽歷史紀錄，但保留您的個人設定與規則。


### 🎨 個人化設定


- **深色/淺色模式**：隨時切換介面風格以適應您的瀏覽環境。
- **自訂分類**：將任何網站分配至適合您工作流程的分類中。


## 🚀 安裝教學


1. 下載此專案程式碼。
2. 開啟 Chrome 瀏覽器並前往 `chrome://extensions/`。
3. 開啟右上角的 **開發者模式 (Developer mode)**。
4. 點擊 **載入未封裝項目 (Load unpacked)**。
5. 選擇包含此擴充功能的資料夾（含有 `manifest.json` 的資料夾）。


## 📖 使用指南


1. **固定擴充功能**：點擊 Chrome 的拼圖圖示並固定 **LocalLog**，以便快速存取。
2. **彈出視窗**：點擊圖示可查看今日簡報及切換專注模式。
3. **儀表板**：點擊彈出視窗中的「儀表板與設定」進入完整的分析介面。
4. **設定規則**：前往「分類與限制」分頁，即可新增網域、設定時間限制或調整網站分類。


## 🛠️ 技術細節


- **Manifest V3**：採用最新的 Chrome 擴充功能標準，安全性更高、效能更好。
- **高效能設計**：使用記憶體緩衝區（Memory Buffer）減少對硬碟的寫入頻率（每 5 秒或切換分頁時存檔），降低資源消耗。
- **無外部依賴**：使用純 JavaScript、HTML 和 CSS 構建，輕量且快速。


## 🔒 隱私權政策


我們非常重視您的隱私：


- **無追蹤代碼**：我們不使用 Google Analytics 或任何第三方追蹤服務。
- **本地儲存**：您的瀏覽紀錄永遠只保留在您的電腦中。
- **權限宣告**：我們僅請求執行功能所需的最小權限（分頁、儲存、通知）。
